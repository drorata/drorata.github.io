The importance of order

  • Tue 14 November 2017
  • HowTo
  • #python, #pandas

Consider the following simply table:

df = pd.DataFrame(
    {
        "cat": [1,1,1,3,2,2,2,3],
        "val": [1,2,3,4,3,2,5,1]
    }
)

and the following three different grouping by:

df.groupby('cat')['val'].apply(tuple)
df.sort_values('val').groupby('cat')['val'].apply(tuple)
df.sort_values('val', ascending=False).groupby('cat')['val'].apply(tuple)

If you'd try it out, you will find that each yields a different order in the generated tuple for each group. This should not come as a surprise, but one should be aware of it!
